## 🖥️ Projeto

Esse é um projeto web responsivo de uma capsula do tempo para exibir memorias em um linha do tempo.

## 🚀 Tecnologia

Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
